```

```


## 📦 Installation

1. clone the repo

```
git clone https://github.com/amiof/lvim.git ~/.config/lvim

```
2.that is better cache reset :
```
lvim +LvimUpdate +LvimCacheReset +q
```

3. use lvim :)
```
lvim
```
